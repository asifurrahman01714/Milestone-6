### To move backward through history:
```bash
history.back()
```
### you can move forward 
```bash
location.href
or
history.forward()
```
### Moving to a specific point in history
```bash
window.history.go(-1)
or
window.history.go(1)
```

### And see this article for details
```bash
https://developer.mozilla.org/en-US/docs/Web/API/History_API
```
